This program is for the 4 DOF 3D printed robot - [TortoBot](http://makerlab.cs.hku.hk/index.php/en/65-tortobot-stem-robot-kit). The oscillator library is taken from [here](https://github.com/JavierIH/octosnake).

This version implements the [GoBLE protocol](https://www.dfrobot.com/product-1276.html) to control the TortoBot, download the iOS app [GoBLE app](https://apps.apple.com/us/app/goble-bluetooth-4-0-controller/id950937437) to play with it.
